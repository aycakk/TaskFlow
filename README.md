# TaskFlow Backend

TaskFlow Backend is the Spring Boot service layer of the TaskFlow System project. It provides REST endpoints, authentication support, and backend-side task management logic for the Android client.

## Features

- REST API for task operations
- Authentication endpoints
- Token-based security support
- Validation and structured request/response handling
- Database integration with JPA / Hibernate
- OpenAPI / Swagger support

## Technologies

- Java 17
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- PostgreSQL
- Maven
- Swagger / OpenAPI

## Purpose

This backend was built to support a more scalable task management architecture by moving core task operations and authentication flows into a dedicated service layer.
